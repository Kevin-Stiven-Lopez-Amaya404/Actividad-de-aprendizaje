## 🧭 ¿De qué trata este proyecto?

> Este repositorio documenta la **actividad de aprendizaje** desarrollada durante la certificación de **GitHub Actions**, con énfasis en la automatización de pipelines de integración y despliegue continuo (**CI/CD**).

A lo largo del proyecto se implementaron flujos de trabajo que permiten **compilar, probar y desplegar APIs funcionales** de forma automatizada, integrando además una base de datos estructurada y gestionada desde el pipeline.
